# $SWIM — Someone Who Isn't Me

Anonymous green-text token site prototype for SWIM.

## Start locally

```powershell
cd swim-token-site
npm install
npm run dev
```

Open http://localhost:3000

## Current build

- Next.js App Router
- TypeScript
- Tailwind
- Framer Motion
- CRT green terminal homepage
- Anonymous archive / threads / token utility concept blocks
